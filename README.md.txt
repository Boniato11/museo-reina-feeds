# 🎨 Feed RSS - Museo Reina Sofía

Feed RSS automático con las exposiciones temporales del Museo Nacional Centro de Arte Reina Sofía.

## 📡 URL del Feed
https://boniato11.github.io/museo-reina-feeds/exposiciones-reina-sofia.xml
## 🔄 Actualización

El feed se actualiza automáticamente cada **lunes a las 09:00 UTC** mediante GitHub Actions.

## 🛠️ Tecnología

- Python + BeautifulSoup4 para scraping
- GitHub Actions para automatización
- GitHub Pages para hosting

## 📋 Contenido

Cada item incluye:
- Título de la exposición
- Fechas (si están disponibles)
- Descripción breve
- Enlace a la web oficial

## 🔍 Suscribirse

Añade la URL del feed a tu lector RSS favorito (Feedly, Inoreader, etc.)
